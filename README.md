# イベント予約システム セットアップ手順

すべて**無料・Googleアカウントのみ**で完結します。

```
学生が予約フォームに入力 → Googleスプレッドシートに自動で1行追加される
```

| サービス | 用途 | 費用 |
|---|---|---|
| Googleスプレッドシート | 予約データの保存・確認 | 無料 |
| Vercel | サイトの公開 | 無料 |

---

## STEP 1 ── Googleスプレッドシートと連携する

### 1-1. スプレッドシートを新規作成する

1. https://sheets.google.com を開く
2. 「＋ 新しいスプレッドシート」をクリック
3. タイトルを「イベント予約」などに変更しておく

### 1-2. Apps Script を設定する

1. スプレッドシートの上部メニュー「拡張機能」→「Apps Script」をクリック
2. エディタが開いたら、中に最初から書かれているコードを**全て選択して削除**する
3. `google-apps-script/code.gs` の中身を**全てコピーして貼り付ける**
4. 「💾 保存」ボタンをクリック（または Ctrl+S / ⌘+S）

   > ⚠️ **「▶ 実行」ボタンは押さないでください。** エラーが出ます。保存だけでOKです。

5. 上部の「デプロイ」→「新しいデプロイ」をクリック
6. 左上の「種類を選択」横の歯車アイコン ⚙ をクリック →「ウェブアプリ」を選択
7. 以下のように設定する

   | 項目 | 設定値 |
   |---|---|
   | 説明（任意） | 予約受信 |
   | 次のユーザーとして実行 | 自分 |
   | アクセスできるユーザー | **全員** |

8. 「デプロイ」をクリック
9. 「アクセスを承認」をクリック → Googleアカウントを選択 →「詳細」→「〇〇（安全ではないページ）に移動」→「許可」
10. 表示された **ウェブアプリのURL** をコピーしてメモしておく
    （例: `https://script.google.com/macros/s/AKfycb.../exec`）

    > ✅ **動作確認**: コピーしたURLをブラウザで開いて「予約システムのApps Scriptは正常に動作しています。」と表示されればOKです。

---

## STEP 2 ── Vercel でサイトを公開する

> Vercelアカウントを持っていない場合は https://vercel.com でGoogleアカウントを使って無料作成できます。

### 2-1. ファイルをアップロードする

1. https://vercel.com にログイン
2. 「Add New Project」をクリック
3. 画面左下の「**Upload**」（ZIPアップロード）を選択
4. このZIPファイルをそのままアップロード
5. 「Deploy」をクリック（設定変更は不要）
6. 数分後にサイトのURLが発行される

### 2-2. 環境変数を設定する

Vercelダッシュボードでプロジェクトを開き、
**Settings → Environment Variables** に以下の3つを追加する。

| 変数名 | 設定する値 |
|---|---|
| `ADMIN_PASSWORD` | 自分で決めたパスワード（例: `MyPass2025`） |
| `JWT_SECRET` | 64文字以上のランダムな文字列（下記参照） |
| `GOOGLE_APPS_SCRIPT_URL` | STEP 1-2 の手順10でコピーしたURL |

**JWT_SECRET の作り方**（どちらか簡単な方で）
- **方法A（簡単）**: https://it-tools.tech/token-generator を開いて「Length: 64」で生成したものをコピー
- **方法B（コマンド）**: ターミナルで `node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"` を実行

### 2-3. 再デプロイする

**Deployments タブ** → 最新のデプロイ右の「**⋯**」→「**Redeploy**」をクリック。

---

## STEP 3 ── 学生に URL を共有する

```
https://【プロジェクト名】.vercel.app/
```

このURLをメールやQRコードで学生に共有してください。
`/admin` のURLは社内のみで使用し、学生には共有しないでください。

---

## 予約が届いたら

学生が予約を送信すると、STEP 1 で作ったスプレッドシートに自動で1行追加されます。
スプレッドシートを開くだけで最新の予約一覧を確認できます。

---

## 日程を追加・変更するとき

1. `【URL】/admin` を開いてパスワードでログイン
2. 「イベント管理」→「＋ 日程を追加」→ 日付・時刻・会場・定員を入力して保存
3. 即座に予約フォームへ反映されます

---

## パスワードを変更するとき

Vercel → Settings → Environment Variables → `ADMIN_PASSWORD` を編集 → Redeploy
